## Research Skill

When conducting research, follow this workflow:

1. **Identify sources.** Before fetching, list 2-3 URLs that are likely to answer the question. Prefer official documentation, reputable sources, and primary data over secondary commentary.

2. **Fetch and read.** Use `fetch_url` to retrieve each source. Read the content carefully before moving to the next.

3. **Cross-reference.** If sources disagree, note the discrepancy. If a source is inconclusive, fetch another.

4. **Synthesize.** Combine findings into a clear, factual summary. Don't just concatenate — integrate.

5. **Cite.** Mention which source each key fact came from. If you couldn't verify something, say so explicitly.

6. **Be honest about gaps.** If you couldn't find a definitive answer, say what you looked for and what was missing.

Never claim something you didn't find in the fetched content. If a fetch fails, note it and try an alternative source.
