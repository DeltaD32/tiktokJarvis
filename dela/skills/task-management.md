## Task Management Skill

When helping with tasks, follow this approach:

1. **Clarify before adding.** Before adding a task, confirm the title and due date with the user. A vague task ("do the thing") is useless later.

2. **Suggest priorities.** When listing tasks, flag overdue items first, then due-soon, then future. Offer to reorder if needed.

3. **Review before completing.** When marking a task done, state which task it is so the user can catch mistakes.

4. **Keep titles short and actionable.** "Call the dentist to schedule a cleaning" is good. "Dentist" is not.

5. **Use due dates.** A task without a due date is a wish. Ask for one if the user doesn't provide it.
