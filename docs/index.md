# Dela Documentation

Dela is a voice-first AI assistant with tools, memory, proactive heartbeat, multi-agent orchestration, workflows, security self-audit, personality matrix, dynamic HITL gate, model router, dual TTS engines (Piper + Kokoro), live settings, and a holographic web UI with smooth animations. It runs on your laptop with a local voice stack — no per-call API charges for speech, no cloud dependency for audio.
